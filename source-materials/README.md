# Source Materials

This directory stores original course materials such as reports, TeX sources, figures, slide decks, and document files. These files are preserved for archival and authoring purposes and are not imported by the Astro site build.

Public website downloads and images must live in `public/projects/<slug>/`. Keep those public copies in place even when an identical original exists here, because the website relies on their stable `/projects/<slug>/...` URLs.
